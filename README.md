gtfsimport
==========

CLI tool to import GTFS feeds from .zip to PostgreSQL database.

# Usage
<!-- usage -->
```sh-session
$ npm install -g gtfsimport
$ gtfsimport COMMAND
running command...
$ gtfsimport (-v|--version|version)
gtfsimport/0.0.1 linux-x64 node-v13.7.0
$ gtfsimport --help [COMMAND]
USAGE
  $ gtfsimport COMMAND
...
```
